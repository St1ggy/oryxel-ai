# Oryxel Icon Guidelines

- Use `viewBox="0 0 24 24"` unless the icon requires different geometry.
- Prefer `fill="none"` + `stroke="currentColor"` for line icons.
- Keep line aesthetics consistent: rounded caps and joins, balanced negative space.
- Default to subtle stroke weight (`1.6`-`1.8`) and avoid visually heavy corners.
- Let consumers control size and color via `class` (`size-*`, `text-*`) and theme tokens.
- Keep decorative fills minimal and ensure readability in both light and dark themes.
